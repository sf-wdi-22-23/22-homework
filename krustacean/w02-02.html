    1. Client: A client is a program setup to request data from from any computer acting as a server.
    2. Server: A server is any computer that is set up to receive requests from a client and return some form of data.
    3. API:An API is a program running on a server setup to handle requests from clients and return relevant or useful data. They can be as simple as returning UTC or as complex as you could ever want them to be.
