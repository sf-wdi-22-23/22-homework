What is the difference between a constructor and a prototype?

A constructor is a pointer, a reference to the object. It allows you to create replicate copies of an object. The constructor is set by object, so changing an object constructor property will only change that specific object constructor.

A prototype is a property of a function objects that no other type of objects have in their property. They share their properties with their objects.
