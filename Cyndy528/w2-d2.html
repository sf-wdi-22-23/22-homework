Examples of Input

  • text area: where the user can put in any text such as a suggestion box.
  • drop-down lists: where the user can select an item from a category
  • Submit: where the user can select this button to have their information collected and processed.

Definitions

  • Server: a location in which API are processed and collected.
  • API: the information that is entered from a user
  • Client: Forwards all requests to the server
