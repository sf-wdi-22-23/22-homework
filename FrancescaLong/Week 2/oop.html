  1. What is the difference between a constructor and a prototype?

    • A constructor is a method called at the moment an object is instantiated. It usually has the same name as the class containing it. It is a pointer, or reference, to the fuction that created it.

    • A prototype is the parent of an object. An object inherits all of its properties from its parent. Javascript is a prototype language.
