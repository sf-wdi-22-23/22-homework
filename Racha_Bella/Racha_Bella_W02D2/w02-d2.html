Three Inputs type

  1. CheckBox : Choosing one or many options of kinds of appartment (single room, double room , shared room
  2. File input : When we want to upload an image or a document
  3. Textarea : When we are answering to an essay question for an application form

Definitions

  1. Client

    2. The client is the one who is asking for something(data) (request) to the server

  2. Server

    3. The server is the one who is reponding to that request (sending data) if it founds them

  3. API

    4. An API makes the communication between the client and the server easier
